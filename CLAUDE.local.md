# Local Project Overrides

## GitHub Workflow

- **Upstream:** `moltis-org/moltis` (read-only, no push)
- **Fork:** `Cstewart-HC/moltis` (push access, ADMIN)
- **Workflow:** Fix on fork → open PR against upstream main. Never push directly to upstream.
- **Default branch:** `main` on both.
- **Issue tracking:** GitHub Issues on `Cstewart-HC/moltis` (enabled).
